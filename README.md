🚗 Safe Drive 360 — Secure Black Box Network

A Secure Vehicle Incident Monitoring & Evidence System that captures accident data, encrypts it, and stores it in a tamper-proof blockchain-style evidence chain for insurance, legal, and safety analysis.

📌 Overview

Safe Drive 360 is a real-time accident detection and evidence management platform designed for:

🚗 Fleet Monitoring
🚑 Accident Detection
🔐 Tamper-Proof Evidence Storage
📡 Real-Time Device Monitoring
⚖️ Court-Ready Evidence Reports
🛡️ Insurance Fraud Prevention

The system captures video, GPS, telemetry, and impact data and securely uploads it within seconds.
